Quantum-Secure Image Transmission

Entanglement protocol with chaos encryption and eavesdropping detection.

Chaos-Assisted QKD Using Entanglement

This project builds an end-to-end E91 quantum key distribution system for secure image transmission. The workflow includes entangled Bell pairs, random basis measurement, CHSH-based eavesdropping detection, secure key extraction, and chaos-assisted encryption.

Highlights

  • Implemented E91 QKD circuits in IBM Qiskit.
  • Achieved near-perfect differential randomness metrics in image encryption experiments.
  • Published in IET Quantum Communication.
