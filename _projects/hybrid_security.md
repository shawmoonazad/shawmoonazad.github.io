## QKD, Classical Cryptography, and Steganography

This project combines E91 QKD, AES-256 encryption, hashing, and CNN-based steganography for multilayered secure image communication. It demonstrates how quantum key establishment can complement deployable classical security layers.

### Highlights

- Built a production-oriented hybrid quantum-classical security pipeline.
- Combined IBM Qiskit quantum simulation with PyTorch-based deep learning steganography.
- Published in **Alexandria Engineering Journal**.
