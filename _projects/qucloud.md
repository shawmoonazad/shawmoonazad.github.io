## Quantum-Safe Cloud Storage

QuCloud studies secure cloud storage through a hybrid design that combines quantum key distribution, post-quantum cryptography, and custom proxy re-encryption. The work addresses quantum-safe sharing and access control in cloud environments.

### Highlights

- Combines QKD-derived trust anchors with post-quantum cryptographic primitives.
- Uses proxy re-encryption for secure data sharing in cloud settings.
- Published in **Journal of Information Security and Applications**.
